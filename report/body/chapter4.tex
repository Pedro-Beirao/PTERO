\section{Workplan}
\Cref{fig:workplan} outlines the planned work for this dissertation. The schedule is organized into three main phases: Experimentation, Code Development, and Deliveries. During the Experimentation phase, there will be a focus on studying the standard and running experiments to establish a solid technical foundation. The Code Development phase overlaps partially with Experimentation, ensuring that the software evolves in parallel with the experimental insights. Finally, the Deliveries phase spans the entire project duration and includes writing the dissertation and preparing the final presentation, ensuring that all outputs are documented and communicated effectively.
